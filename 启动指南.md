## 🚀 快速启动（3步完成）

### 第 1 步：复制 CNN 模型文件

```bash
cp ./rel-react/models/model_INT8.onnx ./rel-react-refactored/backend/models/
```

### 第 2 步：配置环境变量

```bash
cd ./rel-react-refactored
cp .env.template .env
# 编辑 .env 文件，配置 Firebase 和至少一个 AI API
```

### 第 3 步：启动服务

**后端**（终端 1）：
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

**前端**（终端 2）：
```bash
cd frontend
npm install
npm run dev
```

访问：**http://localhost:5173**
