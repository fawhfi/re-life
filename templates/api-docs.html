Re-Life
Data API

Read the latest published, pre-sorted recycling amount for each region with one public JSON request.

01 / Quick start

One endpoint. No input.

Send a GET request and accept JSON. Authentication is not required for this public aggregate dataset.

GET https://relifeapp.com/api/data

No query parameters or request body. Location, GPS, IP address, region filters and custom sorting are intentionally rejected. The response always contains the complete published dataset in database order.

03 / Response format

A stable JSON contract.

A successful request returns HTTP 200. Amounts are expressed in tonnes and records are already ordered from highest to lowest recycled amount.

{
  "data": [
    {
      "region": "Region A",
      "recycled_amount": 12540.5
    },
    {
      "region": "Region B",
      "recycled_amount": 9031.0
    }
  ],
  "meta": {
    "period": "2025",
    "unit": "tonnes",
    "source": "Published source name",
    "source_url": "https://data.example.org/dataset",
    "sorted_by": "recycled_amount",
    "sort_order": "descending",
    "record_count": 2
  }
}
data[].regionstringPublic region name.
data[].recycled_amountnumberNon-negative recycled amount for the region.
meta.periodstringReporting period shared by the whole dataset.
meta.unit"tonnes"Measurement unit for every amount.
meta.source_urlHTTPS URLPublic provenance for the published dataset.
meta.record_countintegerNumber of records returned in data.

04 / Errors

Predictable failure modes.

Errors are JSON objects with a stable machine-readable error value.

400DATA_REQUEST_INPUT_NOT_ALLOWEDThe request included a query string or body.
406JSON_RESPONSE_REQUIREDThe Accept header explicitly rejects JSON.
429Rate limitToo many requests were sent in a short period.
503DATASET_UNAVAILABLENo validated published dataset is currently available. Retry after the indicated interval.
