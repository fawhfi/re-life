## 🚀 现在可以开始部署

### 方式1: 一键自动部署（推荐）

**Windows用户**:
```bash
deploy-all.bat
```

**Mac/Linux用户**:
```bash
chmod +x deploy-all.sh
./deploy-all.sh
```

### 方式2: 分步部署

**第1步**: 阅读文档
```bash
START_HERE.md  # 从这里开始
```

**第2步**: 部署后端到Railway
```bash
cd backend
npm install -g @railway/cli
railway login
railway init
railway up
```

**第3步**: 部署前端到Cloudflare
```bash
cd frontend
npm install -g wrangler
wrangler login
./deploy-cloudflare.bat  # Windows
./deploy-cloudflare.sh   # Mac/Linux
```
